# SG-Unlearn
* Install packages: 
  - numpy 
  - scipy
  - pytorch
  - cvxpy
  - cvxpylayers
  - scikit-learn
  - memory_profiler
 
* Run the code
  - first run `SG_run.sh`; this generates the unlearned model and the corresponding forget, retain, test, validation sets.
  - then run `baselines_run.sh` under `UnlearnSparse` folder.
